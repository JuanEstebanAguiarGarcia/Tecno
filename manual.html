Variables:

Char: Es un tipo de datos integral, lo que significa que el valor se almacena como un número entero, diseñado para el almacenamiento de letras.

String: Es una variable que almacena una secuencia de letras u otros caracteres, como "Que pasa..." . Al igual que los otros tipos de datos.

Int: Almacena numeros enteros, pero NO con decimales, puede ser el 5 o el -5.

Bool: Almacena valores con dos estados, ya sean verdaderos o falsos.

Float: Una variable de tipo Float es una variable que puede contener un número real, como 5.002, -8.77 o 0.03.

Vectores:

Permiten almacenar muchos valores en posiciones de memoria de manera contínua, lo que conyeba que se pueda acceder a un valor de forma eficaz y rápida.

Librerias:

Iostream: Para poder tener acceso a los dispositivos estándar de entrada o salida.

Time: Permite tener acceso al reloj interno del computador.

Stream: Permite utilizar cadena de caracteres.
